# Phase 4.1 — Provider Model Catalog and Selection

Phase 4.1 extends the provider layer with explicit model catalog and selection behavior without altering earlier lifecycle, release, or job contracts.

## Phase deliverables

- provider model catalog contract and schema
- runtime catalog storage under `.audiagentic/runtime/providers/`
- deterministic model selection (explicit id, alias, default)
- catalog refresh CLI
- provider documentation updated with model guidance
