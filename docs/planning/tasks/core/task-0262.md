```yaml

    line: 42
    compatible_with_events: true
    replacement_event: planning.item.state.changed
    migration_priority: high

    file: .audiagentic/planning/config/hooks.yaml
```

1. High priority: hooks that map cleanly to events
2. Medium priority: hooks needing minor adaptation
3. Defer: hooks that can't use events (pre-commit, etc.)
